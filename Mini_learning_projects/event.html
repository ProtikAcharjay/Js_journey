Type: The type of event that occurred. For example, click, mouseover, or keydown.

Timestamp: The time at which the event occurred, in milliseconds since the Unix epoch.

defaultPrevented: A Boolean value indicating whether the default action for the event has been prevented. For example, if you prevent a click event on a button, the button will not be clicked.

target: The DOM element that the event occurred on.

toElement: The DOM element that the mouse pointer is over when the event occurs.

srcElement: The DOM element that initiated the event. This property is deprecated and should be avoided.

currentTarget: The DOM element that the event listener is attached to.

clientX: The horizontal coordinate of the mouse pointer, relative to the document window.

clientY: The vertical coordinate of the mouse pointer, relative to the document window.

screenX: The horizontal coordinate of the mouse pointer, relative to the screen.

screenY: The vertical coordinate of the mouse pointer, relative to the screen.

altKey: A Boolean value indicating whether the Alt key is pressed.

ctrlKey: A Boolean value indicating whether the Ctrl key is pressed.

shiftKey: A Boolean value indicating whether the Shift key is pressed.

keyCode: The code of the key that was pressed.
